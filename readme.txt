1. This application can be run on chrome only.
2. Both a chrome application as well as source code are attached.
3. The source code can be run as a file itself. It is not required to be run as local host.
4. The game can be hosted on localhost and played on phone.
5. The source code can be put on phone and game can be played as is.

INSTRUCTIONS:
On the Computer:
1. The keys 4 and 6 or A and D are used to control movement. 4 moves the boat to the right while 6 moves boat to left.
2. The aim of the game is to navigate through the icebergs. The game ends when you collide with the icebergs or the edge of the screen.
3. Try to cross as many icebergs as possible.
